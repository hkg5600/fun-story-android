## 펀스토리 안드로이드 앱
<br>

### 프로젝트 설명

모두 작가가 되어 자신의 이야기를 공유하는 SNS

### 사용 기술

* Clean Architecture <br> 
* MVVM <br> 
* RxJava2 <br> 
* Refrofit2 <br> 
* Material Design <br> 
* Room <br> 
* Koin
